# 古典文法クイズ

`index.html` をそのまま GitHub Pages に置けば公開できます。

## 使い方

1. GitHub リポジトリのルートに `index.html` を配置します。
2. GitHub Pages を有効にします。
3. ブラウザで開くと、古典文法のクイズアプリとして動きます。

## 収録内容

- 動詞
- 形容詞
- 形容動詞
- 助動詞
- 助詞

## 追加しやすい場所

- `QUESTIONS` 配列に問題を足すだけで出題を増やせます。
- `POS_OPTIONS` や `PARTICLE_TYPES` を調整すれば選択肢を拡張できます。# Classical-Grammar-Quiz
